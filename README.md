An enterprise-grade, fully type-safe software ecosystem engineered from scratch to streamline remote technician tracking and automate administrative billing workflows.

### 🏗️ Architecture & Features
* **Full-Stack Framework:** Built using **Next.js** and **TypeScript**, leveraging Server Components for optimized data fetching and strict type safety across the entire application lifecycle.
* **Database & Type-Safe ORM:** Architected a robust relational database in **MariaDB**, utilizing **Prisma ORM** to handle automated schema migrations, complex data relationships, and compile-time query validation.
* **Remote Technician Workflows:** Engineered a mobile-responsive interface tracking real-time arrival timestamps, precise travel metrics, work order data entries, and live task-status toggling (`Completed` vs. `Work in Progress`).

* **Admin Business Intelligence Dashboard:** Created a comprehensive control panel using Next.js layout structures, allowing administrators to live-monitor field updates, eliminate data silos, and resolve billing inquiries immediately.
* **Backend Security:** Secured the entry points using server-side environment configurations (`.env`) and cryptographic password hashing.
